# Python HRMS 功能說明（SQLite 版）

## 系統設定
- `config.json`：`db_path`、`export_path`、`default_lang`（預設日文）、`version`（0.3.1）；db_path 更新後需重啟。
- i18n：`strings_zh.json` / `strings_ja.json` / `strings_en.json`，主畫面與各子畫面共用詞條。

## 主畫面
- 語言切換（繁中/日/英）、字級調整、全螢幕。
- 登入後載入權限，按鈕依權限可見；狀態列顯示版本、登入帳號與 DB 路徑。

## 主要子畫面與功能
- 員工主檔：員工資料 CRUD、篩選；多語表頭。
- 個人資訊：聯絡/緊急聯絡 CRUD；多語表頭。
- 學歷/畢業資訊：CRUD、篩選；多語表頭。
- 證照設定：證照資料 CRUD；部門下拉來自部門維護；多語表頭。
- 證照-機台對應：證照/機台對應 CRUD、篩選；多語表頭。
- 訓練/證照紀錄：新增前查重，重複即更新並提示；CSV 匯出；多語表頭。
- 到期檢核：依天數篩選逾期紀錄；多語表頭。
- 區域/部門/職務維護：代碼/名稱 CRUD；多語表頭。
- 權限管理：帳號/密碼(MD5)/各功能許可勾選，表格同步顯示；按權限控制主畫面按鈕；可設定 DB 路徑並建立新 DB（預設 HRMS_Database.db），更新後需重啟。
- 報表/匯出：CSV 匯出，欄位與標頭支援多語（持續增補）。

## 資料與匯出
- SQLite `./data/hrms.db`，可用 `scripts/load_sample.py` 產生示範資料與 admin/pigo（密碼 md5(admin123)）。
- 訓練紀錄匯出 CSV，若失敗需提示；其他匯出項目持續擴充。
