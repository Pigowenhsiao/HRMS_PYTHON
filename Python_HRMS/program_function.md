# Python HRMS 功能說明（SQLite 版）

## 後端
- SQLite 資料庫 `./data/hrms.db` 為主要儲存；DAO 採參數化查詢，預留 MySQL 適配。
- 設定檔（config.json/.env）管理 DB 路徑、匯出路徑、版本號、權限設定；多語詞條採集中資源檔。

## 主畫面（main_screen.py）
- PyQt5 QMainWindow，分區入口：員工資料、證照與訓練、管理與權限、報表與分析；內建語系切換（繁中/英文/日文）。

## 員工模組（Python 實作目標）
- 基本資料：員工編號、姓名、部門、職稱、到職日、班別、區域、功能別、啟用。
- 個人資訊：性別、生日、聯絡方式、緊急聯絡人、年資、備註、啟用。
- 學歷檢視：學歷、畢業學校、科系，支援條件篩選；所有欄位標籤支援多語顯示。

## 證照/訓練模組（Python 實作目標）
- 證照設定：部門/區域、證照代碼、名稱、時數、等級、類型、備註、啟用。
- 證照-機台對應：證照對應 Tool_ID，維護備註與啟用。
- 訓練/證照紀錄：員工、證照、日期、等級、類型、啟用、Remark、流水號；新增/修改後可匯出 CSV（匯出訊息支援多語）。
- 到期檢核：計算距證照日期天數，提供到期清單與篩選。

## 權限模組（Python 實作目標）
- 帳號啟用/停用；依登入者（OS 帳號或應用登入）控制按鈕顯示與操作鎖定。
- 啟動流程：讀取設定檔中的版本/DB 連線；驗證 `authority` 表是否允許；未授權則隱藏管理/匯出/敏感按鈕。
- 預設管理員：建立 `admin` 帳號（密碼 `admin123`，建議部署時改為雜湊並修改初始密碼）。

## 報表與匯出（Python 實作目標）
- 訓練報表：依部門/天數篩選並匯出 CSV；與權限掛鉤，僅授權者可執行匯出；報表標題/欄位支援多語。
- 自訂查詢/匯出：以條件組合輸出檔案；匯出失敗改寫本機並提示，記錄日誌供重試；輸出檔案可選語系。
- 多語資源：所有 UI/匯出提示/報表欄位使用同一詞條來源，以便一致翻譯與維護。
