# Python HRMS 開發計畫（SQLite 版）

## 架構與技術
- 桌面：PyQt5（多語、字級調整、全螢幕），集中 i18n JSON（繁中/日/英）。
- 儲存：sqlite3，DB 路徑在 `config.json`；保留切換 MySQL 的適配空間。
- 分層：UI ↔ DAO，設定/匯出路徑由 config；權限與登入檢查於主畫面後。
- 打包目標：PyInstaller，隨附 SQLite 與設定檔；提供 MySQL/網路路徑範例。

## 版本路線
- v0.3.1（現行）：多語全面套用、權限可見控制、部門/區域/職務維護、證照設定部門下拉、訓練紀錄查重自動更新、版本顯示。
- 後續：
  - 報表/匯出欄位選擇與多語表頭。
  - 權限 UI 同步（表格勾選與下方勾選一致），批量啟用/停用。
  - 完整測試與打包。

## 主要模組
- 主畫面：語言/字級/全螢幕；登入權限決定按鈕可見；版本顯示同步 config。
- 員工/個資/學歷：CRUD、篩選、多語表頭。
- 證照設定：部門下拉(來自部門維護)、多語表頭與欄位。
- 機台對應：多語表頭與欄位，篩選含證照/機台。
- 訓練/證照紀錄：新增前查重，重複則更新並提示；CSV 匯出。
- 到期檢核：多語表頭、可輸入天數篩選。
- 區域/部門/職務維護：多語表頭與欄位，權限控管。
- 權限管理：帳號/密碼(MD5)/各功能勾選，表格勾選顯示。
- 設定檔：`config.json`（db_path/export_path/default_lang/version）。
